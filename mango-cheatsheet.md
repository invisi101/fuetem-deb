# MangoWC Keybind Cheatsheet

## Apps
| Keybind | Action |
|---------|--------|
| Super + Enter | Kitty terminal |
| Super + Space | Rofi launcher |
| Super + E | Nautilus file manager |
| Super + Z | Yazi (in Kitty) |
| Super + B | Brave |
| Super + C | Chromium |
| Super + V | LibreWolf |
| Super + N | Standard Notes |
| Super + T | Mousepad |
| Super+Alt + X | Zen Browser |
| Super+Alt + M | Brave PWA (music) |
| Super+Alt + P | Brave PWA |
| Super+Alt + Y | FreeTube |
| Super+Alt + E | Rofimoji (emoji picker) |
| Super+Alt + V | Clipboard (rofi) |
| Super+Alt + L | Swaylock |

## Window Management
| Keybind | Action |
|---------|--------|
| Super + W | Close window |
| Super + F | Toggle maximize |
| Super+Shift + F | Toggle fullscreen |
| Super + S | Toggle floating |
| Super + M | Send to scratchpad |
| Super+Shift + M | Restore from scratchpad (floating) |
| Super + P | Toggle scratchpad |
| Super + G | Toggle global (sticky) |
| Super + Y | Toggle overlay |
| Super + X | Toggle overview |
| Super + O | Focus next in stack |

## Focus (arrows or vim keys)
| Keybind | Action |
|---------|--------|
| Super + H / Left | Focus left |
| Super + L / Right | Focus right |
| Super + K / Up | Focus up |
| Super + J / Down | Focus down |

## Move Window (arrows or vim keys)
| Keybind | Action |
|---------|--------|
| Super+Shift + H / Left | Move left |
| Super+Shift + L / Right | Move right |
| Super+Shift + K / Up | Move up |
| Super+Shift + J / Down | Move down |

## Tags (Workspaces)
| Keybind | Action |
|---------|--------|
| Super + 1-9 | Switch to tag |
| Super+Shift + 1-9 | Move window to tag |
| Super+Ctrl + 1-9 | Toggle tag on window |
| Super + [ | Previous tag |
| Super + ] | Next tag |
| Super+Shift + [ | Move window to previous tag |
| Super+Shift + ] | Move window to next tag |

## Layout
| Keybind | Action |
|---------|--------|
| Alt + Space | Cycle layout |
| Alt + E | Scroller: set proportion to 100% |
| Alt + X | Scroller: cycle proportion preset (50%/80%/100%) |

## Monitor
| Keybind | Action |
|---------|--------|
| Super + , | Focus left monitor |
| Super + . | Focus right monitor |
| Super+Shift + , | Send window to left monitor |
| Super+Shift + . | Send window to right monitor |

## Floating Window
| Keybind | Action |
|---------|--------|
| Ctrl+Shift + Arrows | Move floating window |
| Ctrl+Alt + Arrows | Resize floating window |

## Gaps
| Keybind | Action |
|---------|--------|
| Alt+Shift + X | Increase gaps |
| Alt+Shift + Z | Decrease gaps |

## Screenshots & Tools
| Keybind | Action |
|---------|--------|
| Print | Screenshot (Satty) |
| Ctrl + Print | Full screenshot |
| Super+Shift + C | Color picker (hyprpicker) |
| Super+Shift + S | Screensaver |

## Media Keys
| Keybind | Action |
|---------|--------|
| Volume Up/Down | Volume +/- 10% |
| Mute | Toggle mute |
| Mic Mute | Toggle mic mute |
| Play | Play/pause |
| Stop/Prev/Next | Media controls |
| Brightness Up/Down | Brightness +/- 10% |

## Mouse
| Keybind | Action |
|---------|--------|
| Super + Left click drag | Move window |
| Super + Right click drag | Resize window |
| Middle click (overview) | Toggle maximize |

## Scroll Wheel
| Keybind | Action |
|---------|--------|
| Super + Scroll up | Next occupied tag |
| Super + Scroll down | Previous occupied tag |

## Touchpad Gestures
| Gesture | Action |
|---------|--------|
| 3-finger swipe | Focus window in direction |
| 4-finger left/right | Previous/next occupied tag |
| 4-finger up/down | Toggle overview |

## System
| Keybind | Action |
|---------|--------|
| Shift+Alt + R | Reload config |
| Super+Shift + E | Quit MangoWC |
| Ctrl+Alt + Delete | Quit MangoWC |
